# projeto-imersao-dev
A Biblioteca de Tatuagens é um catálogo visual interativo que reúne estilos, origens e referências da arte corporal. Com cards dinâmicos, busca inteligente e visual moderno, o projeto permite explorar técnicas como blackwork, realismo, oldschool e muito mais. Um espaço criado para inspirar artistas e apaixonados por tatuagem.
